# Learning-Raja-Adaptive-AI-Learning-Agent-
AI-powered adaptive learning platform with personalized tutoring, quizzes, analytics, and gamification built using Streamlit and OpenAI.
It personalizes learning by generating topic-specific quizzes, tracking student performance, and providing real-time analytics through a visually engaging dashboard.
The platform introduces gamification (XP points, streaks, badges, celebrations) to motivate learners while ensuring structured academic progression through quizzes, mock tests, and final assessments
